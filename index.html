产品灵感

AI无处不在，高效交互

当AI能力触手可及，日常交互将从「多步操作」变为「一句话搞定」。我们识别了三大高频场景痛点，用语音智能一一化解。

生活服务

点咖啡、叫外卖、取快递、打车、线上购物——每一件都需要打开APP、多步点击、反复确认。

痛点：打开外卖APP → 搜索店铺 → 选商品 → 加购物车 → 选地址 → 确认支付，至少6步操作
OneTouch：按键说「帮我点一杯瑞幸生椰拿铁，送到公司」→ AI自动完成下单

工作服务

会议中灵感冒出却来不及记录，待办事项散落在各个工具里，跨平台沟通效率低下。

痛点：打开钉钉 → 找到对话 → 输入消息 → 发送，或切换到备忘录手动记录再同步
OneTouch：按键说「提醒我下午3点给张总发项目方案」→ 自动创建待办并同步钉钉

个人助手

好想法转瞬即逝，日程管理碎片化，邮件信息应接不暇，任务优先级难以厘清。

痛点：灵感来了掏出手机 → 解锁 → 找APP → 打字记录，等准备好灵感已消失
OneTouch：按键说「记一下，产品方案要加入语音唤醒功能」→ 即刻存入灵感库
生态架构

应用与开发者生态

四层架构连接硬件到用户，开放的Skill生态让更多开发者参与共建，为C端用户提供无限丰富的应用场景。

1

设备层 · OneTouch 硬件

Push-to-Talk 按键、TTS Speaker、WiFi/CAT1 双模通信、电子墨水屏

2

平台层 · IoT云 + AI PaaS

设备管理、OTA升级、多模型调度引擎、A2A/CLI协议网关、语音ASR/TTS服务

3

应用层 · Skill 市场 SaaS

Skill路由引擎、服务编排、开发者SDK、第三方接入管理、用量与计费

4

用户层 · 小程序 / APP

Skill选择与激活、个性化配置、使用记录、多设备绑定、家庭组管理

开发者生态

让更多Skill和服务开发者加入，共同构建丰富的语音交互应用生态。

开放 Skill SDK

提供标准化Skill开发框架，支持Python/Node.js，包含语音意图解析、A2A协议对接、CLI调用等核心能力。

开发者认证体系

分级认证（个人 / 企业 / 战略合作），提供技术支持、测试沙箱和优先推荐位。

收益分成模型

按Skill调用次数计费，开发者获得70%收益分成，平台收取30%服务费。头部Skill享额外流量激励。

社区与生态治理

开发者社区论坛、Skill质量评级、用户反馈闭环、安全审核机制，确保生态健康发展。

核心功能

三端协同，全链路智能化

从端侧硬件到云端服务，再到用户管理界面，每一层都经过精心设计，提供无缝的语音交互体验。

端侧 · 硬件能力

Push-to-Talk 按键唤醒

物理按键一键唤醒AI，零延迟启动语音采集，避免语音唤醒词的误触发问题。

TTS Speaker 语音播报

内置扬声器实时播报AI反馈，语音合成自然流畅，支持音量自适应调节。

WiFi + CAT1 双模连接

WiFi适合室内场景，CAT1蜂窝网络覆盖户外移动使用，自动切换保证连接稳定。

电子墨水屏 (Pro版)

1.54寸 e-ink 显示屏，超低功耗显示时间、状态、Skill图标和简要反馈信息。

180天超长续航

低功耗芯片架构，日均交互10次可续航180天以上，USB-C磁吸充电。

极致小型化

目标尺寸 60×60×18mm，约65g，可放置桌面、粘贴墙面或随身携带。

服务侧 · 云端能力

A2A 协议网关

Agent-to-Agent协议，支持智能体间通信，实现跨Skill协同与复杂任务编排。

CLI 服务调用

通过命令行接口直连钉钉、飞书、企业微信等平台，完成会议预约、消息发送等操作。

多模型调度引擎

智能路由至最优AI模型，支持大语言模型、语音模型、NLU模型的灵活配置和切换。

Skill 路由引擎

基于语音意图自动匹配最佳Skill，支持上下文感知、多轮对话和Skill间无缝切换。

OTA 远程升级

固件与Skill配置空中升级，设备始终保持最新能力，无需用户手动操作。

用户侧 · 管理体验

小程序 / APP 管理

通过微信小程序或独立APP管理设备，界面简洁直观，上手即用。

Skill 市场

一站式浏览、搜索、试用和激活各类Skill，按类别、评分和热度排序。

个性化配置

自定义唤醒提示音、默认Skill偏好、语音语速、TTS音色，打造专属交互风格。

多设备绑定

一个账号管理多台设备，支持桌面、卧室、车载等不同场景的设备分组。

使用数据统计

可视化查看交互频次、Skill使用分布、活跃时段，了解自己的AI使用习惯。

Phase 2 · 开发者版

开放Skill SDK和开发者平台，接入钉钉/飞书CLI，CAT1网络支持，Skill市场上线，开发者认证启动。

Phase 3 · 消费者版

两款SKU量产上市，完整Skill生态，家庭组管理，多设备联动，全国渠道铺设和品牌推广。
